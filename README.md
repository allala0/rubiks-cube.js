<h1 align="center">Rubiks Cube</h1>
<p align="center">Rubiks cube of any size in browser.</p>
<p align="center"><img src="img/cube.png" width=300;/></p>

## How to use

After downloading this repo using ```git clone https://github.com/allala0/rubiks-cube.js``` you need to run this on server, can be local server for example: XAMPP or you can run server in Python.
